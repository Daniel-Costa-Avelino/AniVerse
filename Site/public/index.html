Objetivo do site

Esse site quer contribuir para que o usuário compreenda sobre o que é Anime e caso crie interesse nesse mundo, possa receber algumas dicas de animes para assistir. Temos como foco também, contribuir para que mais pessoas possam ser impactadas com essa forma de arte que conquista e une pessoas.

Para que o usuário possa usufruir de forma completa as funcionalidades do site, como, a página de Perfil, ele deve se CADASTRAR e LOGAR. Nessa página de perfil, terá informações públicas do usuário e ele poderá adicionar os animes que ele já assistiu e uma dashboard totalmente dinâmica. Está esperando o que?
